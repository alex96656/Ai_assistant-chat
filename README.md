# 🤖 AI Telegram Assistant Bot

A smart Telegram AI assistant bot that replies like ChatGPT using Node.js and OpenAI API.

---

## ✨ Features

- 💬 AI-powered chat responses (ChatGPT style)
- ⚡ Fast and real-time replies on Telegram
- 🧠 Understands natural language questions
- 🔐 Secure API key handling using environment variables
- ☁️ Easy deployment on Railway or any Node.js hosting
- 📱 Works directly inside Telegram (no extra app needed)

---

## 🚀 Tech Stack

- Node.js
- Telegram Bot API
- OpenAI API
- dotenv
